# Bank-Loan-Analysis-for-Finance

**Executive Summary**

Using SQL and Power BI, I analyzed bank loan data to track key KPIs such as total loan applications, funded amount, amount received, interest rate, and debt-to-income ratio. The analysis identified trends in loan performance, credit risk, and repayment behavior, along with month-over-month changes.
The dashboard highlights the distribution of good loans (Fully Paid/Current) vs bad loans (Charged Off), enabling better visibility into portfolio health and risk exposure.
Based on the findings, I recommend the following actions:

1. Strengthen credit risk evaluation to reduce charged-off loans

2. Focus on high-performing loan segments with better repayment behavior

3. Continuously monitor MoM trends to adjust lending strategies

**Business Problem**

Loan performance directly impacts profitability and risk in financial institutions. High default rates and limited visibility into lending trends make it difficult to manage risk and optimize decisions. This project aims to analyze loan data to identify performance patterns and reduce default risk while improving overall portfolio health.

**Methodology**

1. Extracted and analyzed loan data using SQL queries

2. Calculated key KPIs such as total applications, funded amount, received amount, interest rate, and DTI

3. Performed month-over-month (MoM) analysis to track trends and performance changes

4. Segmented data by loan status, state, term, purpose, and home ownership

5. Built interactive dashboards in Power BI to visualize insights and enable filtering

**Skills**
1. SQL: Aggregations, Joins, Filtering, KPI Calculations
   
2. Power BI: Dashboard Design, DAX, Data Modeling, Data Visualization

3. Data Analysis: KPI Tracking, Trend Analysis, Financial Analysis

**Results and Business Recommendation**

The analysis was conducted on 38,576 loan applications, with a total funded amount of $435.76M and total amount received of $473.07M, indicating overall positive returns on issued loans. The average interest rate is 12.05%, and the average debt-to-income (DTI) ratio is 13.33%.
The portfolio shows that 86.17% of loans are good (Fully Paid/Current), while 13.82% are bad loans (Charged Off), highlighting a manageable but important level of credit risk.
